# NOFA Daily Task Manager

A simple single-file daily task management app built in `index.html`.

This tool helps organize daily work with task tracking, priorities, statuses, due dates, focus blocks, notes, and browser-based saving using `localStorage`.

## Features

- Add, edit, and delete tasks
- Set task priority:
  - High
  - Medium
  - Low
- Set task status:
  - To Do
  - In Progress
  - Done
- Assign category and focus block
- Add due dates and notes
- Search tasks quickly
- Filter by status and priority
- View dashboard stats
- Live date and time display
- Automatic browser saving with `localStorage`

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser `localStorage`

No backend is required for this version.

## Project Goal

This app was created as a lightweight daily work organizer for internal use. It is designed to be fast, simple, and easy to improve brick by brick.

The first version is intentionally kept minimal so it can run anywhere as a single file.
